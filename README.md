# Zami AI Studio

Automatización para crear influencers virtuales latinas con IA.
Genera rostro → cuerpo → perfil de personalidad completo desde un servidor local.

## Inicio rápido

```powershell
cd C:\Users\LENOVO\zami-ai-studio
git pull origin claude/virtual-influencer-face-gen-ZDwju
.\iniciar.bat
# Abrir http://127.0.0.1:3333
```

## Configuración

```bash
cp .env.example .env
# Llenar VITE_COMFYDEPLOY_API_KEY y ANTHROPIC_API_KEY
```
